# SOP Supervisor — MyProdusen

Panduan ini untuk Supervisor. Supervisor fokus pada tim sendiri dan tidak memiliki akses default ke payroll, setting Superadmin, atau data tim lain.

## 1. Login

1. Buka `/login`.
2. Masukkan email/username dan password.
3. Klik masuk.
4. Pastikan dashboard tim terbuka.

## 2. View Team Dashboard

1. Buka dashboard.
2. Cek ringkasan anggota tim.
3. Cek kehadiran tim hari ini.
4. Cek notification terkait leave, KPI, atau attendance.

## 3. View Team Attendance

1. Buka Attendance.
2. Gunakan filter tanggal jika perlu.
3. Cek status anggota tim.
4. Laporkan data aneh ke Admin HR.
5. Jangan mencoba akses data tim lain.

## 4. Approve / Reject Leave

1. Buka Leave.
2. Pilih request dari anggota tim.
3. Baca tanggal dan alasan.
4. Klik approve jika valid.
5. Klik reject jika tidak valid.
6. Isi alasan rejection secara jelas dan sopan.
7. Pastikan notification terkirim ke employee.

## 5. Rejection Reason Rule

- Alasan wajib diisi saat reject.
- Alasan harus spesifik, bukan hanya “tidak bisa”.
- Gunakan bahasa profesional.
- Jangan mencantumkan komentar pribadi yang tidak relevan.

## 6. Input / Review Team KPI

1. Buka KPI.
2. Pilih anggota tim.
3. Input hasil KPI sesuai data kerja.
4. Review skor sebelum submit.
5. Approve jika role dan workflow mengizinkan.
6. Jangan mengubah KPI approved tanpa proses resmi.

## 7. View Team Performance

1. Buka KPI atau dashboard performance.
2. Cek performa per employee.
3. Catat employee yang perlu coaching.
4. Gunakan data untuk diskusi internal dengan HR.

## 8. View Notifications

1. Buka Notifications.
2. Cek permintaan baru.
3. Tandai sebagai read setelah diproses.
4. Follow up request yang belum selesai.

## 9. Yang Tidak Bisa Diakses Supervisor

- Tidak bisa akses dashboard Superadmin.
- Tidak bisa manage semua user.
- Tidak bisa melihat data tim lain.
- Tidak bisa akses payroll by default.
- Tidak bisa download PDF report Superadmin.
- Tidak bisa mengubah setting sistem.
- Tidak bisa melihat selfie employee di luar tim.
