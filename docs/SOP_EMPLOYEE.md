# SOP Employee — MyProdusen

Panduan ini untuk karyawan. Gunakan MyProdusen untuk absensi GPS + selfie, pengajuan izin/cuti/sakit, melihat KPI, payroll sendiri, dan notifikasi.

## 1. Register Account

1. Buka `/register`.
2. Isi nama, email, username, dan password.
3. Klik daftar.
4. Tunggu email aktivasi.
5. Jika email tidak masuk, cek folder spam atau hubungi HR.

## 2. Activate Account From Email

1. Buka email aktivasi dari MyProdusen.
2. Klik tombol aktivasi.
3. Pastikan halaman aktivasi berhasil.
4. Login setelah aktivasi.

## 3. Login

1. Buka `/login`.
2. Masukkan email/username dan password.
3. Klik masuk.
4. Jika akun belum aktif, hubungi HR.

## 4. Check-In Dengan GPS + Selfie Realtime

1. Login dari HP Android asli.
2. Buka menu Attendance.
3. Izinkan akses lokasi.
4. Izinkan akses kamera.
5. Pastikan berada di area kerja.
6. Ambil selfie realtime dari kamera.
7. Klik check-in.
8. Tunggu status berhasil.

## 5. Check-Out Dengan GPS + Selfie Realtime

1. Buka menu Attendance.
2. Pastikan sudah check-in hari itu.
3. Izinkan lokasi dan kamera jika diminta.
4. Ambil selfie realtime.
5. Klik check-out.
6. Tunggu status berhasil.

## 6. GPS / Camera Permission Troubleshooting

- Jika GPS ditolak, buka pengaturan browser dan izinkan lokasi.
- Jika GPS tidak akurat, pindah ke area terbuka lalu coba lagi.
- Jika kamera ditolak, buka pengaturan browser dan izinkan kamera.
- Jika selfie gagal, refresh halaman dan pastikan kamera tidak dipakai aplikasi lain.
- Jika tetap gagal, hubungi HR dengan screenshot error.

## 7. View Attendance History

1. Buka Attendance History.
2. Pilih tanggal atau periode.
3. Cek jam check-in, check-out, status, dan catatan.
4. Laporkan data tidak sesuai ke Supervisor/Admin HR.

## 8. Submit Leave / Sick / Permission

1. Buka Leave.
2. Pilih jenis pengajuan: cuti, sakit, atau izin.
3. Isi tanggal dan alasan.
4. Upload lampiran jika diminta perusahaan.
5. Submit.
6. Tunggu approval Supervisor/Admin HR.

## 9. View KPI

1. Buka KPI.
2. Cek KPI yang assigned ke akun sendiri.
3. Cek nilai dan status approval.
4. Jika data tidak sesuai, hubungi Supervisor.

## 10. View Payroll Own Data

1. Buka Payroll.
2. Cek periode payroll sendiri.
3. Download payslip jika tersedia.
4. Jika payroll tidak tampil, hubungi HR.
5. Karyawan tidak bisa melihat payroll karyawan lain.

## 11. View Notifications

1. Buka Notifications.
2. Baca update leave, KPI, attendance, atau pengumuman.
3. Tandai read jika sudah dibaca.

## 12. Change / Reset Password

1. Untuk ganti password saat login, buka Profile atau Password.
2. Isi password lama dan password baru.
3. Simpan.
4. Jika lupa password, buka `/forgot-password`.
5. Ikuti link reset dari email.

## 13. Logout

1. Klik menu akun.
2. Klik logout.
3. Pastikan kembali ke halaman login.
4. Jangan tinggalkan akun login di HP bersama.
