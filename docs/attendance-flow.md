# Attendance Mobile Flow

MyProdusen attendance now supports mobile-first Clock In/Clock Out with shift schedule, geofence validation, realtime selfie, success page, today log, and Superadmin schedule management.

## Employee Flow

```text
Dashboard Karyawan
↓
Shift Aktif card
↓
Clock In / Clock Out
↓
Langkah 1 dari 2: Validasi lokasi GPS + map preview
↓
Langkah 2 dari 2: Selfie realtime mirror + catatan opsional
↓
Submit
↓
Absensi Berhasil + Log Absensi Hari Ini
```

## Main Screens

- `/dashboard` shows employee today shift card.
- `/dashboard/attendance` shows detail absen, active shift, Clock In/Out button states, and today log.
- `/dashboard/attendance/clock?type=clock-in` validates GPS before selfie.
- `/dashboard/attendance/clock?type=clock-out` validates GPS before selfie.
- `/dashboard/attendance/success` shows submission result and today timeline.
- `/dashboard/attendance/schedules` lets Superadmin assign per-day employee schedules and valid locations.

## Backend Endpoints

- `GET /api/attendance/today`
- `POST /api/attendance/validate-location`
- `POST /api/attendance/check-in`
- `POST /api/attendance/check-out`
- `GET /api/attendance`
- `GET /api/attendance/selfie/:path`
- `GET /api/attendance/schedules?date=YYYY-MM-DD`
- `POST /api/attendance/schedules`
- `DELETE /api/attendance/schedules/:id`
- `GET /api/attendance/shift-locations/:shiftId`
- `PUT /api/attendance/shift-locations/:shiftId`

## Data Model

Additive migration `0031_employee_schedules_multi_location.sql` adds:

- `Shift.isSpecialShift`
- `EmployeeSchedule`
- `ScheduleLocation`
- `ShiftLocation`
- `Attendance.scheduleId`

No destructive operation. No `db:push` needed.

## Security Rules

- All attendance endpoints require auth.
- Backend revalidates geofence on submit.
- Location from frontend is never trusted alone.
- Selfie is required for Clock In/Out.
- No gallery upload flow exists.
- Selfie endpoint is protected.
- Employee/Leader can submit only own attendance.
- Superadmin manages schedules and valid locations.
- Audit log remains generated by attendance service.
